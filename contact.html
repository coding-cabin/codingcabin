General

I would like to learn how to code; do you guys know of any resources? How/Where did you guys learn how to code?

Most of our members learned to code through a combination of Codecademy, W3Schools, and tweaking existing Tumblr themes. After that, we consult Google and Stack Overflow when we have questions, run into problems, or wish to try something new.

Information

Can I join? How do I join?

There are 2 requirements:

  1. You must have at least one code released. It may be a theme, page, update tab, etc. There is no quality standard it must meet. Make sure we can find it.
  2. Reblog the Coding Cabin introduction post.

Send us a message asking for a link to join the server if/when you’ve met both!

UPDATE (6/11/2020): You can now join Coding Cabin without having released any themes yet. View this post to find out more.

I want to join but I haven't released any themes of my own yet. Can you make an exception?

UPDATE (6/11/2020): You can now join Coding Cabin without having released any themes yet. View this post to find out more.

We’re so happy to know you’d like to join our server!

Unfortunately, our main requirement is to have at least one code released. We hope you understand why we can’t open exceptions; if we do it once, we’d be obligated to grant entry for anyone who asked, even if they don’t meet the criteria we set for the server.

We’re looking forward to seeing your first code though! We hope you’ll join us then!

What is the dynamic of the server like? How active are the members, and how active are members expected to be?

The dynamic in Coding Cabin’s server is quite good, though. We’re a group of very friendly people and we try to welcome every member the same way. We don’t differentiate between long-term members and new members; if you’re in Coding Cabin, you’re one of us even if you’re there for only ten minutes!

There are very different ages as well, so there’s big respect when it comes to how active you are. For example, if you have a family to take care of or if you simply need some time away, we totally understand and we won’t be angry or judge you for it. Even if you aren’t in any of these situations and simply don’t feel like talking, that’s alright too!

You can find a detailed answer here.

Do members make collaboration themes here, or is this more of just a network.

This is more of a network, but as with all networks, members may team up to release collaborative themes.

Coding Awards

What are the Coding Awards?

Coding Cabin's Coding Awards is a competition organized by suiomi and annasthms with the goal to promote creativity and web-design good-practices through fun challenges

This competition will be on-going throughout the year and it will be made of fun coding challenges that everyone is welcome to participate in!

How can I participate?

  1. Reblog the challenge's post.
  2. Either submit your theme to us or tag it with #codingawards and/or the challenge’s tag.
  3. On your entry's post, make sure to mention that it was made for this competition and link it to the Coding Awards’ informational page.

How do I know if my entry has been received?

If we have received your entry, it will show up on this page. If a period of time has passed since you posted your entry and it is not on the page, double check that all the participation requirements have been met and submit it to us again. If you are unsure if the requirements have been met, feel free to message us for help.

When will the winners of the ____ challenge be announced?

We will try to announce the winners within a week of the challenge’s end date.

Can only members win?

Anyone can participate and anyone’s got a chance to win!

Our challenges are not exclusive to Coding Cabin members!
